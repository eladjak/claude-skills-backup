# Creative AI Process

## When to Use
Use when doing AI-assisted creative work - content creation, design, writing, brainstorming. Prevents rushing to solutions.

## Framework: Depth Before Speed

### Phase 1: Immersion (Don't Skip!)
- Study 10+ examples of the target style/genre
- Identify patterns, not just surface features
- Ask "why does this work?" not just "what does it look like?"
- Document the underlying principles

### Phase 2: Constraints
- Define what you're NOT doing (equally important)
- Set quality criteria before generating
- Establish brand/style guardrails
- Decide on target audience and context

### Phase 3: Exploration
- Generate 5+ variations, not just one
- Push each variation to its extreme
- Mix unexpected combinations
- Don't evaluate yet - just create

### Phase 4: Refinement
- Now evaluate against Phase 2 criteria
- Combine best elements from different variations
- Iterate on details, not wholesale changes
- Get feedback from target audience if possible

### Phase 5: Polish
- Final quality check against original brief
- Verify consistency across all elements
- Test in real context (device, platform, size)
- Document what worked for future reference

## Anti-Patterns
- Accepting the first AI output as final
- Skipping the analysis/immersion phase
- Not defining constraints before creating
- Optimizing too early (polish before exploring)
- Working in isolation without reference material
