# Command Palette

Unified access to all Claude Code capabilities.

## Quick Access

| Command | Description |
|---------|-------------|
| `/orchestrate` | Show all available commands |
| `/orchestrate skills` | List skills |
| `/orchestrate agents` | List agents |
| `/orchestrate recent` | Recently used |
| `/orchestrate run <cmd>` | Execute command |

## Categories

- **Development**: git, build, test, deploy
- **AI Agents**: SAGE, planners, reviewers
- **Utilities**: backup, sync, search
- **Reports**: daily summary, stats

## Tips

- Type partial names to fuzzy-search
- Most used commands appear first
- Combine with skill-search for discovery
